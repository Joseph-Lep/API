Vous allez retoucher votre fichier index.php (à la racine du projet) pour qu'il agisse en tant que routeur :
Il devra réceptionner l'URI et le verbe HTTP envoyés par le client (allez voir du côté de la superglobale $_SERVER)
En fonction des valeurs réceptionnées, il devra exécuter l'opération appropriée
Par exemple
si le routeur reçoit GET /courses, il renverra une liste au format JSON de tous les cours (toutes les ressources courses)
si le routeur reçoit GET /courses/5, il renverra un seul élément course au format JSON
